GitHub学生包申请指北

Jul 14, 2022

GitHub学生包到期，续期时发现无论怎么上传证件都无法成功，最终通过以下方式+1。

资料准备

有几个关键点需要在上传的证件中包含，缺少一项则失败概率up+++

  • 入学时间
  • 预计毕业时间
  • 就读学校
  • 姓名
  • 照片

学信网在线学籍认证截图是一个很好的样例

可以使用浏览器自带翻译把它转成英文（不转也行）

Google机翻效果图如下

操作步骤

进入网址

Students - GitHub Education

选择学生包

填写院校信息

下面这一步很重要，决定了跳转页面的证件上传方式。

❌不可在填写学校时选择GitHub候选框中的学校，自行填写校名，中英文皆可。

完成学校和理由申请信息后下一步。

上传学生证件照片

右下角 Upload an image 上传证件，后面正常填入学校信息。

🎉恭喜，提交成功！

同时邮箱会收到一封邮件，大约一周（一般三天内）会返回审核结果。

最后

如选择GitHub下拉框中默认学校，会在跳转时进入GitHub的默认检验流程。该流程没有图片上传选项，只能使用摄像头拍摄，极难通过验证（但是如果核验成功应该是秒过）

本人尝试在不使用科学上网的情况下（本地IP）使用虚拟摄像头截取英文版学信网截图（高清图片），无法通过验证。
